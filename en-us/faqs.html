FAQs

1. Where is the GitHub repository for this website?

https://github.com/chenyuan33/chenyuan33.github.io

2. Can I contribute to this website?

Yes, we welcome contributions. You can submit suggestions or code via in Pull Requests or Issues or the comment section of any page. You can mail to me if you don't want to make your suggesstions public.

3. I link the dark theme. Can I switch the theme?

Of course, you can. Click the sunny Emoji icon in the upper right corner to switch between the light and dark themes. If you like, you can also switch between the light and dark themes repeatedly.

4. I am not a person using this language. How to switch the language?

Of course, you can use the translation function of your browser. Additionally, this website has built-in languages. You can find the description of the current language at the top of the page (such as English), and click it to switch.

5. If I have any questions, where can I ask them?

You can ask them in Issues or in the comment section of any page. You can mail to me if you don't want to make your questions public.
