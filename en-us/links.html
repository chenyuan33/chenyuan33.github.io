How to add a link

A way

  1. Fork this repository.
  2. Edit the file en-us/links.html.
  3. Add a new line after the last <li> tag.
  4. Add a new <li> element like this: <li><a href="[your link]"><img src="[your avatar]" alt="[your name]" />[your name]</a></li>
  5. Commit and push the changes to your fork.
  6. Go to your link and add a link to this website.
    name: cy3's site
    avatar: https://chenyuan33.github.io/favicon.png
    link: https://chenyuan33.github.io/
  7. Create a pull request to this repository.

Another way

Go to your link and add a link to this website.
