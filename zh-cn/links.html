如何添加友链

方法一

  1. Fork 本仓库。
  2. 编辑文件 zh-cn/links.html。
  3. 在最后一个 <li> 标签后添加一行。
  4. 在 <li> 元素中添加如下内容： <li><a href="[你的链接]"><img src="[你的头像]" alt="[你的名字]" />[你的名字]</a></li>
  5. commit 并 push 你的修改到你 fork 的仓库。
  6. 前往你的链接并添加一个友链。
    名字：cy3's site
    头像：https://chenyuan33.github.io/favicon.png
    链接：https://chenyuan33.github.io/
  7. 将你 fork 的仓库 pull request 到本仓库。
