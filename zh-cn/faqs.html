常见问题

1. 本网站的 GitHub 仓库在哪里？

https://github.com/chenyuan33/chenyuan33.github.io

2. 我可以对这个网站做出贡献吗？

当然，我们欢迎。你可以通过 Issues 或任意页面下方的评论区提出改进建议或使用 Pull Requests 提交代码。如果你不想公开你的建议，你也可以向我发送邮件。

3. 我喜欢暗色主题，可以切换到暗色主题吗？

当然可以。点击右上角的“太阳” Emoji 图标即可切换，如果愿意，你还可以在明亮主题和暗色主题之间反复切换。

4. 我不使用这个语言，我应该如何切换语言？

当然，你可以使用浏览器的翻译功能。此外，这个网站了内置一些语言。你可以找到顶部的描述当前语言的文字（如 English），并点击它进行切换。

5. 如果我有其他问题，应该在哪里提问？

你可以在 Issues 中提出你的问题，或者在任意页面下方的评论区中提出你的想法。如果你不想公开你的问题，你也可以向我发送邮件。
